# HangoutYuk Vote API / Poll Website

### This repo contains the web template and web framework required to create a static polling website for creating polling places
### Templating is done using [Eta.js](https://www.npmjs.com/package/eta) template engine
### This app is deployed using GAE 

## Library used

1. [express](https://www.npmjs.com/package/express)
2. [mysql2](https://www.npmjs.com/package/mysql2)
3. [nanoid@3.3.4](https://www.npmjs.com/package/nanoid)
4. [sequelize](https://www.npmjs.com/package/sequelize)
5. [axios](https://www.npmjs.com/package/axios)
6. [dotenv](https://www.npmjs.com/package/dotenv)
7. [cors](https://www.npmjs.com/package/cors)
8. [http-status](https://www.npmjs.com/package/http-status)
9. [eta](https://www.npmjs.com/package/eta)
